Detailed Description

COMMAND MACROS.

Bulk packets will always be sent in the order of the definiton of the macros E.g. |SEND_BULK|VEL|ACCEL|JERK|L_DISP_X|L_DISP_Y|U_DISP_X|U_DISP_Y|

All packets must be 6 bytes in length. Where data is not needed, stuff with dummy bytes.

See the related API documentation for further information.
